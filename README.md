# iData

iData is a index page for data science. 

## :battery: Dependencies

- **jQuery**
- **sass**
- **iconfont** 
- **ajax** - to fetch json data
- **art-template** - page rendering
- **echo.js** - image lazy loading
- **nprogress** - page loading progress bar
